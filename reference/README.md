# Reference sketches

These are illustrative code sketches showing the intended structural
patterns. They are **not** the final implementation and have known
divergences from the specs.

Each sketch file has a header comment listing what it's missing.
Additional divergences discovered during the design review are documented
in the relevant spec's "Notes" section and in `DECISIONS.md`.

Use these for the *shape* of the code, not as copy-paste starting points.
